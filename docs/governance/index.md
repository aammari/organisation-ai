# Gouvernance

Documentation des règles de gouvernance de l'Organisation AI.

## Workflow State Machine

Les états projet sont définis dans `core/workflow_engine.py`.

## Exception Reports

Tout écart déclenche un `ER-XXXX` via `core/deviation_service.py`.

## Decision Governance

Les décisions sont tracées avec `DEC-XXXX` via `core/decision_service.py`.
